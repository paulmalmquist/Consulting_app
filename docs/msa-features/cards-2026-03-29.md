### Development Pipeline Spatial Map
WeHo's brief surfaced 7 active pipeline projects clustered around GEODIS Park stadium, but there was no way to visualize their spatial relationship to this infrastructure catalyst. The map overlay would show project markers color-coded by status with infrastructure catalyst radius rings (0.25mi, 0.5mi, 1mi). Nearly all zones have development pipelines (frequency 8/10), making this broadly useful. Medium build effort — needs geocoding + mapping integration.

### Opportunity Zone Capital Flow Tracker
WeHo contains OZ-designated census tracts but actual capital flows into those tracts were untraceable via public search. The existing OZ Boundary Overlay card (priority 15) maps which parcels are OZ-eligible, but this new card tracks whether OZ incentives are actually driving investment. Would use proxy signals (new LLC formations, construction permit dollar volumes) since IRS Form 8996 aggregate data is not publicly available at tract level. Lower frequency (5/10) since not all zones contain OZ tracts, and harder to build (effort 3/10).
