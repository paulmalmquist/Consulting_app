# Coding Session Report — 2026-03-22

## Task Selected
**CRITICAL: repe_fast_path empty dashboards** — Analytical queries like "Show me top 5 assets by NOI" and "Plot NOI over time" were producing empty dashboard shells with 0 tokens, 0 tools, breaking 3 of 6 AI test cases (Tests 2, 3, 5 in the 2026-03-21 AI test report).

## Root Cause
`_CHART_INTENT_RE` in `repe_intent.py` matched chart-related language ("top N", "NOI over time", "bar chart") and boosted `INTENT_GENERATE_DASHBOARD` to score 0.90. This then suppressed `INTENT_ANALYTICS_QUERY` via the suppression list at line 685. The dashboard handler (`compose_dashboard_spec`) only creates layout specs without fetching data — producing empty shells that the frontend rendered as "COMPLETED, 1 widgets • Custom • 2 KPI(s)" with no actual content.

## What Changed

### Commit 1: `e6c9f0a` — Intent classification fix
**File: `backend/app/services/repe_intent.py`**

1. **Lowered chart-keyword dashboard score** from 0.90 to 0.65 — chart keywords alone no longer exceed the 0.70 suppression threshold. Explicit "dashboard"/"report" keywords still score 0.90.

2. **Boosted analytics for chart queries** — When chart keywords match but no explicit "dashboard" keyword appears, `INTENT_ANALYTICS_QUERY` gets boosted to 0.88. This routes data queries through the SQL agent, which actually fetches data.

3. **Fixed suppression logic** — `INTENT_GENERATE_DASHBOARD` only suppresses analytics when the dashboard score came from an explicit dashboard/report request, not from chart keywords.

**Scoring before fix:**
- "Show me top 5 assets by NOI" → dashboard 0.90, analytics 0.14 → **dashboard wins (empty)**
- "Plot NOI over time" → dashboard 0.90, analytics 0.00 → **dashboard wins (empty)**

**Scoring after fix:**
- "Show me top 5 assets by NOI" → dashboard 0.65, analytics 0.88 → **analytics wins (fetches data)**
- "Plot NOI over time" → dashboard 0.65, analytics 0.88 → **analytics wins (fetches data)**
- "Build me a portfolio overview dashboard" → dashboard 0.90, analytics 0.00 → **dashboard wins (correct)**
