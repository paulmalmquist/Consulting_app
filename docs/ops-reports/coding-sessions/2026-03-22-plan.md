**CRITICAL: repe_fast_path empty dashboards** — Analytical queries like "Show me top 5 assets by NOI" and "Plot NOI over time" produce empty dashboard shells with 0 tokens, 0 tools.

## Root Cause
`_CHART_INTENT_RE` in `repe_intent.py` matches chart-related language (top N, NOI over time, bar chart, etc.) and boosts `INTENT_GENERATE_DASHBOARD` to 0.90, which then suppresses `INTENT_ANALYTICS_QUERY`. The dashboard handler (`compose_dashboard_spec`) creates layout specs without fetching data, producing empty shells.

## Fix
1. Lower chart-keyword dashboard score from 0.90 to 0.65 (below the 0.70 threshold that suppresses analytics)
2. Boost `INTENT_ANALYTICS_QUERY` score when chart keywords are detected (data-fetching queries need the SQL agent)
3. Remove `INTENT_GENERATE_DASHBOARD` from analytics suppression list when the dashboard score didn't come from explicit "dashboard" keywords
