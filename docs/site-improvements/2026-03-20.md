### 2. Homepage Hero — Fix the Subheading Grammar

**Problem:** The subheading reads: "Consulting engagements to show measurable ROI using AI-native workflows Built for leaders under real pressure to modernize." The capital "B" mid-sentence indicates a lost line break. "To show" is passive.

**Improvement:** Clean the copy:

**Suggested copy:**

```

Execution engagements that deliver measurable ROI — not strategy decks.
Built for operations leaders under real pressure to modernize.
```

**Strategic reasoning:** The first content a buyer reads has a grammar error. Five-minute fix.

### 3. About Page — Name the Founder

**Problem:** "Novendor was founded by operators with deep experience" — but no one is named. The app behind paulmalmquist.com is substantial — 5 environments, 273+ pages, multi-vertical, institutional demo data. The person who built this has credibility to claim. The About page doesn't claim it.

**Improvement:** Add a founder section.

**Suggested copy:**

```
WHO WE ARE

Novendor is led by Paul Malmquist. Background in real estate private equity
technology, fund operations, and enterprise data systems.

The workflows Winston automates — waterfall calculations, LP reporting,
capital call processing, portfolio monitoring — are workflows we've built

```

**Strategic reasoning:** The app itself is the strongest credibility signal Novendor has. A $2B demo portfolio with BlackRock and CalPERS as LPs, multi-fund waterfall execution, and a domain-specific AI copilot — this is not a side project. But the marketing site doesn't connect the product depth to a named builder. One paragraph fixes that.

**Suggested copy:**

```
You're not paying for software. You're paying for organizational avoidance.
```

**Improvement:** Pull one pain-point line into the hero as a secondary hook:

**Suggested copy:**

