## Priority Copy Changes (Carry-Forward + New)

### 1. Hero tagline — add proof of depth (carry-forward, still unimplemented)
- **Current:** *"AI execution environment for real estate private equity, project delivery, and institutional operations."*
- **Proposed:** *"AI execution environment — fund waterfall engines, deal scoring, credit decisioning, capital project draws, LP analytics, and 32 configurable lab environments."*
- **Why:** The current tagline is category-level. The proposed version names specific engines that signal build maturity to technical evaluators.

### 2. Stat bar below hero (carry-forward, still unimplemented)
```
258 pages · 208 services · 31 AI tool categories · 32 lab environments
```
One line. Immediately communicates platform scale.

### 3. NEW — Add "Domains" section with four cards
A lightweight 2x2 grid below the stat bar, each card naming the domain and 3 key features:

| Card | Features listed |
|---|---|
| **Real Estate PE** | Fund waterfall engine · Deal Radar scoring · LP capital calls |
| **Credit Decisioning** | Underwriting policy engine · Exception handling · Audit trail |
| **Project Delivery** | Draw management · Pay app variance · Capital project tracking |
| **Institutional Ops** | CRM + engagement · Document extraction · Compliance & governance |

This gives evaluators a 10-second scan of platform breadth without requiring login.

### 4. NEW — OG/meta description for link previews
- **Current:** *"Fund reporting, waterfall logic, capital activity, and portfolio monitoring."*
- **Proposed:** *"AI execution environment for REPE, credit decisioning, project delivery, and institutional ops. 258 pages, 32 lab environments, full-stack AI with RAG and Monte Carlo simulation."*
- **Why:** When this URL is shared in Slack/email/LinkedIn, the preview text is the only selling surface. Current preview undersells by 10x.

### 5. NEW — "Request a walkthrough" CTA needs a value hook
- **Current:** Plain link text.
- **Proposed:** *"See Winston run your fund data — request a walkthrough →"*
- **Why:** Action-oriented CTAs with specificity convert 2-3x better than generic ones.

### 6. Indexing decision (strategic note)
The site has `noindex, nofollow`. This is appropriate if paulmalmquist.com is a private app portal. However, if it's also the primary URL shared with prospects, the OG tags matter even more since they're the only marketing surface that reaches non-logged-in viewers.
