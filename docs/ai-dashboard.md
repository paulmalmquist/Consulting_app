# AI Dashboard Docs Redirect

The canonical dashboard request docs for Winston live under `docs/dashboard_requests/`.

Use these files instead of inventing new top-level dashboard spec paths:

- `docs/dashboard_requests/README.md`
- `docs/dashboard_requests/template.md`
- `docs/dashboard_requests/schema.md`
- `docs/dashboard_requests/real_estate_fund_dashboard.md`

If an agent asks for `docs/ai-dashboard.md`, treat that as a shorthand pointer to `docs/dashboard_requests/README.md`.
