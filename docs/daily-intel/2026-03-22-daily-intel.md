The combined entity positions real-time data as the backbone for enterprise AI agents and automated workflows across hybrid cloud. **Winston relevance:** Reinforces that real-time data pipelines are becoming table stakes for enterprise AI — Winston's existing RAG and dashboard composition already handle streaming context, but this signals the market expects deeper real-time data integration as a default.
