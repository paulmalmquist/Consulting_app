## Top Gaps (Not in Winston, High Enterprise Value)

1. **NOI Delta Explainer** (Easy build, 3-5 days) — CFOs and asset managers constantly need to explain NOI changes to investors and lenders. Automating this saves hours per property per quarter. High enterprise value because it's a pain point in every quarterly report.

2. **Rent Roll Validator** (Easy build, 2-3 days) — Data quality is a constant concern. Automated rent roll validation catches errors before they propagate into financial models. Acquisition teams waste significant time manually checking rent rolls.

3. **Variance Analysis Agent** (Partial, 1 week) — Budget-vs-actual at the operational level is table stakes for institutional reporting. Winston's UW vs Actual is a foundation but needs the operational budget layer.

4. **Agent STUDIO concept** (Major build, but the concept of user-composable workflows is the trend) — The market is moving toward no-code/low-code AI agent composition. Winston should consider a simplified version.

## Quick Wins (Already Partial — Just Need Assembly)

1. **NOI Delta Explainer** — Winston has P&L data and AI chat. Package a "Why did NOI change?" MCP tool that runs attribution analysis and returns a structured narrative.

2. **Variance Analysis** — Extend UW vs Actual to support operational budget baselines. Add narrative generation.

3. **OM Theses Screener** — Combine document ingestion pipeline with Deal Radar criteria to auto-screen incoming OMs against fund investment thesis.

4. **Model-Agnostic Architecture** — Abstract the AI gateway to support model switching. Winston's architecture likely already supports this at the service level.
