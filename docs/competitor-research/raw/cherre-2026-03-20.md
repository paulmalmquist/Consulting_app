## Product Modules / Capabilities

- **Platform** — Core data management and insights foundation; collects, standardizes, validates, and delivers data
- **Workflows (Data Submission Portal)** — Streamlines and standardizes data collection from third-party vendors; data review and approval workflow
- **Agent STUDIO** — Custom AI agent creation platform; design, deploy, and scale purpose-built AI agents using connected data
- **AI Agent Marketplace** — Curated library of "Work Packages" — coordinated teams of AI agents engineered for specific business outcomes; modular "Action Blocks" combined into end-to-end solutions
- **Connections Marketplace** — 150+ pre-built data connectors across industry applications; 1,000+ business-critical tools beyond real estate
- **Data Fabric** — 3.3B+ addresses in knowledge graph; property characteristics, recorder/deeds, valuations, tax assessments, liens, mortgages, boundaries, demographics, POI data
- **Universal Data Model** — Out-of-the-box standardization model for real estate data
- **Semantic Data Layer** — Semantic model share capability for BI integration
- **Knowledge Graph** — ML-powered property matching across identifiers and geospatial layers
