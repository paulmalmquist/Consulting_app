## Product Modules / Capabilities

- **Glean Search** — Enterprise workplace search powered by AI; indexes all company data across SaaS apps
- **Glean Assistant** — Personal AI assistant; data analysis, deep research, canvas, companion
- **Glean Agents** — Build and manage AI agents via natural language; Agent Builder, Agent Governance, Agent Orchestration, Agent Library
- **Enterprise Graph** — Maps relationships across all organizational data
- **Personal Graph** — Tracks individual work patterns and context
- **Glean Protect** — Single-tenant cloud, permission enforcement, sensitive content safeguarding, agent action validation

## AI / Automation Features

- AI search across 100+ SaaS integrations
- Natural language agent creation and deployment
- Multi-agent orchestration
- "Agentic Engine" — plans and adapts across company context
- Enterprise memory (learns organizational processes over time)
- Deep Research (multi-step autonomous research across company data)
- Permission-aware retrieval (access control baked in)
- Multi-model support: GPT, Gemini, Claude, open source
