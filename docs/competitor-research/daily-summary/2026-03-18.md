## New Capabilities Discovered

- **ARGUS Benchmark Manager** (Altus) — Portfolio-to-market performance benchmarking using the world's largest CRE valuation dataset; attribution analysis by driver; positioned as peer comparison tool
- **ARGUS Taliance** (Altus) — Dedicated fund management module: LP waterfall, capital accounts, fund-level returns — confirms this is a crowded space and Winston needs to sharpen its differentiators here
- **Agentic building data ingestion** (Cambio) — LLM-powered extraction of structured data from PDFs/spreadsheets directly into asset data model; this is the architectural pattern Winston should replicate for operating statements and rent rolls
- **Automated compliance reporting** (Cambio) — GRESB, TCFD, SFDR auto-submission; confirmed LP/regulatory pressure is growing; not a near-term Winston priority but a watch item
- **Enterprise Graph + Agent Builder** (Glean) — Horizontal platform building the "AI layer beneath every enterprise app"; confirmed the generic AI vs. domain AI competitive frame is live and will intensify in 2026

## Top 3 Product Opportunities for Winston

1. **Portfolio Scenario Analysis UI** — Partial → Easy build — Priority 9
   Parameterized stress test runner across all assets simultaneously; Winston has the underlying data; 3–5 day build on existing infrastructure; highest demo value of anything analyzed today

2. **Structured Extraction from Operating Documents** — Partial → Easy-Moderate build — Priority 9
   Extend document pipeline to extract structured financial fields (T-12, rent roll) into asset data model; eliminates manual data entry; directly competitive with Cambio's core data ingestion claim at a fraction of the scope

3. **Attribution Analysis for UW vs. Actual** — Moderate build — Priority 8
   Decompose NOI variance into driver waterfall (occupancy, expenses, revenue mix); extends existing UW vs. Actual data into LP-ready narrative output; highest enterprise value relative to build effort

## Top 3 Demo Ideas

1. **The 60-Second Stress Test** — GP / CFO — Needs parameterized scenario UI (3–5 days)
   Type "stress test fund at cap rate +100bps" → get instant IRR impact by asset and fund

2. **Upload a T-12, Get Your Asset Record Populated** — Asset Manager / CFO — Needs structured extraction (3–5 days)
   Upload operating statement PDF → Winston parses and populates asset record → flags anomaly → runs UW vs. Actual comparison

3. **The Variance Waterfall** — GP / CFO / Asset Manager — Needs attribution decomposition (1–2 weeks)
   Ask "what drove the Q3 NOI variance?" → get waterfall chart by driver → draft LP communication
